### Calculadora de Sistemas Lineares - Método de Gauss-Jordan



#### **Descrição**



Este projeto implementa a resolução de **sistemas lineares** pelo método de *Gauss-Jordan* em Python.

&nbsp; 

A ideia inicial surgiu durante minha graduação em matemática onde testei na prática a minha capacidade de resolver Sistemas Lineares e conhecimento em Excel.
O projeto original foi feito no Excel em 2021, foi adaptada inicialmente em 2022 para aula de Linguagem de Programação II e aprimorada em Python como forma de desafio em linguagem de programação.
Em 2026, retorno ao projeto, desta vez com uma experiência de quem lida com o usuário, para refazer alguns detalhes que possam não ter ficado claro aos interessados.





#### Funcionalidades

* Leitura dos coeficientes e termos independentes de um sistema linear.
* Aplicação passo a passo do **método de Gauss Jordan**.
* Retorno das soluções do sistema (se possível)
